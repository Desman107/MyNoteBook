# stl容器
1. vector, 变长数组
2. string，字符串，substr(),c_str()
3. priority_queue,优先队列,push(),top(),pop()
4. queue,队列，push(),front(),pop()
5. stack,栈,push(),top(),pop()
6. set,map,multiset,multimap，动态维护有序序列
7. unordered，哈希
8. bitset，压位
